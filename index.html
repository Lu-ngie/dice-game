Advanced Dice Game

Game Instructions

  1. Enter the names of the two players.
  2. Click the "Roll Dice" button to roll the dice for both players.
  3. The player with the higher dice roll wins the round.
  4. The first player to win 3 rounds wins the game.
  5. A pop-up message will announce the round winner and the game winner.
