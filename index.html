Game Instructions

  1. Enter the names of the three players.
  2. Click the "Roll Dice" button to roll the dice for all players.
  3. The player with the highest dice roll wins the round.
  4. The first player to win 3 rounds wins the game.
  5. A pop-up message will announce the round winner and the game winner.
